This folder is the source of truth for code that is pasted into Squarespace.

sitewide/header.html
  Paste into: Squarespace → Settings → Advanced → Code Injection → HEADER

headers/*.html
  Paste into: Squarespace → Page Settings → Advanced → Page Header Code Injection

blocks/*.html
  Paste into: Squarespace → Page content → Code Block (on that page)

Rule: if you change anything in Squarespace, update the matching file here.
